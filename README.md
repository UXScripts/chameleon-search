chameleon-search
==================

Chameleon Launcher widget that lets you search the Internet using various search engines.

The latest version of the widget can be found at <http://matthewtole.com/chameleon/search/>.

### Supported Search Engines
* Google
* Google (UK)
* Wikipedia
* IMDB
* Google Images
* YouTube
* Bing
